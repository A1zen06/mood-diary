### Мини-дневник настроения
Простое веб-приложение для отслеживания эмоционального состояния. Позволяет выбирать настроение через emoji, добавлять заметки и просматривать историю записей.

### Основной функционал:
Выбор настроения - 8 вариантов emoji с описанием (счастливый, грустный, злой и др.)

Добавление заметок - текстовое поле для дополнительных комментариев

История настроений - список всех сохраненных записей с датами

Управление записями - возможность удалять записи из истории 

Выбор даты - запись настроения на любую дату
